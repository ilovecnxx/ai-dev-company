# 显性意图解析 - 课程笔记

## 核心要点

1. **指令 = 目标 + 约束 + 验收标准**：每条指令都应拆解为三要素，缺一不可
2. **主动追问是专业表现**：缺失要素时应该追问，而不是猜测
3. **复述确认避免返工**：在执行前先确认理解正确，比返工更高效

## 三要素提取法 (Intent Triad)

### 目标 (Goal)
要做什么？完成任务的核心诉求

### 约束 (Constraints)
限制条件 - 时间、格式、风格、范围等

### 验收标准 (Criteria)
怎样算做好？具体可衡量的标准

## SOP 流程

1. **接收指令** - 完整接收，不急于开始
2. **三要素拆解** - 识别目标、约束、验收标准
3. **缺失检测** - 检查是否有隐含或缺失
4. **追问补全** - 用选项式问题而非开放式
5. **复述确认** - "我的理解是...对吗？"

## 常见陷阱

- 过度解读：指令简单却脑补大量需求
- 不敢追问：怕显得笨，宁愿猜测
- 追问过度：每个细节都问，让 Human 觉得不如自己做
